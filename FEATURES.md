# Skinalo: Feature Specification & Progress Tracker

This document provides a detailed breakdown of Skinalo's features, their current development status, and technical requirements.

## 🟢 Core Utility & MVP (Base Engine)

These features form the foundational "Scanner & Analyzer" engine of the app.

| Feature | Description | Status |
| :--- | :--- | :---: |
| **Instant Label Scanner** | High-accuracy OCR to extract ingredient lists from any product packaging or image. | [ ] |
| **Ingredient Breakdown** | Mapping chemicals to functions (Humectant, Occlusive, Emollient, Active). | [ ] |
| **Safety Check** | Real-time alerts for specific skin sensitivities (e.g., alcohol, fragrance, essential oils). | [ ] |
| **Photosensitivity Alerts** | Flagging AHAs, BHAs, and Retinoids with sunscreen reminders. | [ ] |
| **Basic Skin Profile** | User onboarding to define skin type (Oily, Dry, Combo) and main concerns. | [ ] |

---

## 🔓 Free Tier Features

Utility-focused features available to all users.

- [ ] **Unlimited Individual Scans:** No cap on how many products a user can analyze one-by-one.
- [ ] **Ingredient Dictionary:** A searchable internal wiki explaining ingredients in non-clinical language.
- [ ] **Searchable Knowledge Base:** Quick tips on how to identify certain skin reactions.

---

## ✨ Skinalo Elite (Premium / "Beautify")

Advanced clinical-grade coaching and routine management.

### 🗺️ Personalized Roadmaps
Targeted, time-bound plans for specific goals.
- [ ] **Clear Dark Spots:** A 12-week guidance plan focusing on tyrosinase inhibitors.
- [ ] **Lighten Up:** Brightening protocols using safe, non-toxic alternatives.
- [ ] **Smoothing:** Texture-focused routines for congestion and fine lines.

### 🔄 Dynamic Routine Calibration
A responsive engine that adjusts based on feedback.
- [ ] **Sensitivity Toggle:** If a user reports redness, the app suggests "stalling" actives for 48 hours.
- [ ] **Environment Sync:** Automatically suggests heavier occlusives if local humidity drops below 30%.
- [ ] **UV Index Integration:** Mandatory AM routine changes when UV index is >7.

### 📈 The Progress Tracker
- [ ] **Timeline View:** A digital diary of skin changes with photo logging.
- [ ] **Comparison Slider:** Side-by-side "Before & After" visuals with metadata (date, products used).
- [ ] **Milestone Awards:** Gamified achievements for consistency.

---

## 👨‍👩‍👧‍👦 Family & Ecosystem

Expanding the utility to households and local commerce.

- [ ] **Multi-Profile Support:** Manage routines for children or partners from one account.
- [ ] **Age-Specific Engine:** Filters out harsh actives for infant or elderly skin profiles.
- [ ] **Geo-Shop Finder:** Map integration showing local verified retailers carrying "Safe" products.
- [ ] **Authenticity QR Scanner:** A tool for users to verify product batch codes with partner brands.
